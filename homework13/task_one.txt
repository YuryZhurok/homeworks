Fork
Предусловия:
    - открытый браузер
    - переход по ссылке гитхаба
    - авторизуемся(вводим логин/пароль)
    - открытый репозиторий, с которым хотим сделать операцию форк
Шаги: 
    - нажимаем кнопку форк
    - проверяем создание копии репозитория в нашем профиле
Постусловия:
    - открытый гитхаб
    - вы всё ещё авторизованный пользователь
    - рабочая ветка мэин в вашем репозитории


Закрытие pull request
Предусловия:
    - открытый браузер
    - переход по ссылке гитхаба
    - авторизуемся(вводим логин/пароль)
    Шаги: 
    - открываем пул реквест(созданный из ветки тест в ветку мастер)
    - нажимаем кнопку ребейз и мёрдж
    - подверждаем операцию ребейз и мёрдж
    - проверяем закрытие пул реквеста в ветку мэин
Постусловия:
    - открытый гитхаб
    - вы всё ещё авторизованный пользователь
    - рабочая ветка мэин в вашем репозитории


Изменение BIO
Предусловия:
    - открытый браузер
    - переход по ссылке гитхаба
    - авторизуемся(вводим логин/пароль)
    - нажимаем на иконку с вашим аватаром
    - заходим в профиль
Шаги:
    - нажимаем edit profile
    - добавляем описание в графу BIO
    - нажимаем сохранить изменения
    - обновляем страницу
Постусловия:
    - открытый гитхаб
    - вы всё ещё авторизованный пользователь
    - в вашем описании пользователя добавились изменения в графе BIO и отображаются на профиле
